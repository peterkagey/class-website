### Choice Exercise 6 [≤5]
*Do any number of the following parts, in order.*
It turns out that just like we can talk about two numbers being equivalent $\mathrm{mod}\ m$, we can also talk about two polynomials being congruent $\mathrm{mod}\ f(x)$, where $f(x)$ is a polynomial.

The definition is what you might hope it would be!
<ol>
<li>[2] We say $$p(x) \equiv q(x) \pmod{r(x)}$$ if...<br>
Guess what you think the definition is, then click below to get the definition. Compare and contrast your guess with the actual definition.
<details>
  <summary>Polynomial congruence</summary>
  We say that given three polynomials $p(x), q(x),$ and $r(x)$, $$p(x) \equiv q(x) \pmod{r(x)}$$ if there exists a polynomial $f(x)$ such that $p(x) - q(x) = r(x)f(x)$.
</details></li>
<li>[1] Recall that if we're given two (positive) integers $n$ and $m$ and their decimal representation, it's easy to check $n \equiv m \pmod{10}$: just make sure that their last digits match up. What is the analogous rule when looking at polynomials $\mathrm{mod}\ x$</li>
<li>[1] Prove that $x^2 \equiv -1 \pmod{x^2+1}$.</li>
<li>[1] Notice that you can replace every $x^2$ with a $-1$ when simplifying a polynomial $\mathrm{mod}\ x^2 + 1$. Use this to simplify $x^4 + x^2 + 1 \pmod{x^2 + 1}$.</li>

### Choice Exercise 7 [2 + ≤6]
*Your programming exercise. The second part is worth one point for each 6 minutes you spend on this, up to a maximum of 6.*

In Python, we have "boolean" values, `True` and `False`, and logical operators. For example,

- The operator `&` (equivalently `and`) is a "logical and", so `True & False` (equivalently `True and False`) both return `False`.
- The operator `|` (equivalently `or`) is a "logical or."
- The operator `not` is a "logical not," so `not(True)` returns `False`.
- The operator `^` is a "logical xor."

1. [2] Choose a programming language, determine the logical operators, write a truth table for five (inequivalent) logical expressions, and record the results here. (This doesn't have to have fancy formatting. For example, for `and` you can write the output of `print(True & True, True & False, False & True, False & False)`.)
2. [≤6] We can have a truth table which has three columns recording values, $P, Q,$ and $R$. This means that there are $2^{2^3} = 256$ different logical statements we can make up to logical equivalence. Write a program that outputs a logical statement for each different outcome.
