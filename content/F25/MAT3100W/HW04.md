### Required Exercise 4 [4]
In class on Wednesday, week 5, we started to learn some set theory. We ended just before we got to talk about universal sets and De Morgan's law, so we'll do it here.

**Definition** Given some "universal set" $\mathcal{U}$ (usually implicit) and a set $A \subseteq \mathcal{U}$, the *complement* of $A$ (in $\mathcal U$) is $$A^c = \mathcal{U} \setminus A.$$

It's often clear what $\mathcal{U}$ is from the context, but sometimes it is explicitly specified.

<ol>
  <li>Solve the following three questions about set complements.</li>
  <ul>
    <li>Suppose that $\mathcal{O}$ is the set of odd integers. If $\mathcal{U} = \mathbb{Z}$ (the set of integers), what is $\mathcal{O}^c$?</li>
    <li>Suppose that $A = \\{1, 2, 3, 5, 8\\}$ and the universal set is $\mathcal{U} = \{1, 2, \dots, 10\}$. What is $A^c$?</li>
    <li>Suppose that $B = \\{(x,y) \in \mathbb{R} \times \mathbb{R} \mid x^2 + y^2 > 1 \\}$ and $\mathcal{U} = \mathbb{R} \times \mathbb{R}$. What is $B^c$? (You can describe this either geometrically or in set builder notation.)</li>
  </ul>

  <li>Now prove De Morgan's laws. In both cases, assume that $A$ and $B$ are both subsets of the universal set $\mathcal{U}$.</li>
  <ul>
    <li>$(A \cup B)^c = A^c \cap B^c$.</li>
    <li>$(A \cap B)^c = A^c \cup B^c$.</li>
  </ul>
</ol>

*(To prove that two sets are equal, show that everything in the first set is in the second set **and** prove that everything in the second set is in the first set.)*

<br>
When talking about base-$12$ representations, we often write the number with a subscript $12$. If we write a number with digits $a_k, a_{k-1}, \dots, a_1, a_0$ in base-$12$, then this number is equal to $$a_k12^k + a_{k-1}12^{k-1} + \dots + a_1 12^1 + a_0 12^0.$$ For example, $$
50B_{12} = 5(12^2) + 0(12^1) + 11(12^0) = 731.
$$

### Choice Exercise 11 [4]
We can factor quadratic polynomials by "completing the square".

The idea is that if we want to find the roots of a quadratic equation like $2x^2 - 5x + 6 = 0$, we can
- divide by the leading coefficient to get $x^2 - \frac{5}{2}x + 3 = 0$,
- then subtract off the constant part to get $x^2 - \frac{5}{2}x = -3$,
- then add $\frac{25}{16}$ to both sides to get $$
    \underbrace{\left(x - \frac{5}{4}\right)^2}_{x^2 - \frac{5}{2}x + \frac{25}{16}}=-3 + \frac{25}{16} = -\frac{23}{16}.
  $$
- Now we can solve for $x$: $$\begin{align*} x - \frac{5}{4} &= \pm \sqrt{-\frac{23}{16}} \\\\ x &= \frac{5 \pm \sqrt{-23}}{4}. \end{align*}$$

Now your turn!

2. Do the same process again, but where the quadratic polynomial has coefficients $a, b,$ and $c.$ (You'll have to read between the lines of what I did to determine what to do in the general case.)
