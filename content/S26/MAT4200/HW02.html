    3. Beneath that add this code, which will create some objects in blue defined by taking every single red point, and multiplying it by a matrix whose columns all have norm 1 and are orthogonal to each other. (Matrices with this property define the orthogonal group $O(3)$.) 
    color("blue")
multmatrix([
    [-0.707107, 0.408, 0.57735],
    [ 0.707107, 0.408, 0.57735],
    [ 0,        -0.816, 0.57735]
]) {
    cylinder(20,4,4);
    translate([3.5,0,0]) cube([15,8,8],center=true);
    translate([-1,6,16]) linear_extrude(height = 3) text("p");
}
    Describe in words what the matrix has done. In particular, what happened to the "p"?
  3. Now, we're going to make a green piece which consists of the same pattern, with the cylinder rotated in the direction $(-1, -2, -3)$. 
    color("green")
multmatrix([
    [a11, a12, a13],
    [a21, a22, a23],
    [a31, a22, a33]
]) {
    cylinder(20,4,4);
    translate([3.5,0,0]) cube([15,8,8],center=true);
    translate([-1,6,16]) linear_extrude(height = 3) text("p");
}

Now choose another column of the matrix, and replace it with any unit vector which is orthogonal to the one you just found. Lastly, replace the last unit vector with any unit vector which is orthogonal to both.

Exercise 3 [6]

Recall that my Christmas tree was about $7$ feet tall and $4$ feet in diameter at the base. Parameterize the tree, and plot it in Mathematica.
  1. Set up an integral to compute the surface area of the tree.
  2. It's easier to see ornaments near the front of the tree, so naturally, I have a higher density of ornaments there. Suppose that the density of ornaments is modeled by $4(\theta - \pi)^2$ ornaments per square foot, where $\theta$ is the angle from the front of the tree.
    1. "The zeroth lesson in mathematics is you should always begin by guessing!"1 Imagine a tree of the size described, and then imagine four ornaments per square foot at the front. What would you guess is the number of ornaments on the tree?
    2. Set up an integral to compute the number of ornaments on the tree according to this model. Could this possibly be the number of ornaments I actually had on my tree?
    3. How many ornaments do I have in the $90^\circ$ wedge centered at the front of the tree versus the $90^\circ$ wedge centered at the back?

    5. If I have 90 feet of lights, how far should I space them apart if I want to space them evenly?

Exercise []

In this exercise, you will estimate $\pi$ using cubic Bézier curve to approximate the unit circular arc in the first quadrant. Recall that in class, we estimated $\pi$ with the quadratic Bézier curve \[ \mathbf{c}(t) = (1-t)^2 \begin{pmatrix}1 \\ 0\end{pmatrix} + 2t(1-t) \begin{pmatrix}1 \\ 1\end{pmatrix} + t^2 \begin{pmatrix}0 \\ 1\end{pmatrix} \] and we found that \[ \pi \approx 2 \int_{t=0}^1 \sqrt{8 (t-1) t+4} dt \]

  1. Use Mathematica to plot this curve: 
    ParametricPlot[{(1 - t)^2 {1, 0} + 2 t (1 - t) {1, 1} + t^2 {0, 1}}, {t, 0, 1}]
    Write a guess of what you think will happen if you extend the domain from $t \in [0,1]$ to $t \in [-1,2]$. Now edit the code to change the domain and say what actually happened. Right click to save either (or both) plots as an image, and include them in your writeup with the $\LaTeX$ command \includegraphics[width=\textwidth]{path/to/image.png}.
  2. Use Mathematica to compute this integral with 
    2*Integrate[Sqrt[4+8(-1+t)t],{t,0,1}]
    Highlight the output, and select Copy As > LaTeX, and paste it here.
  3. Wrap the whole expression in an N[...], to get a numerical approximation of the value. Paste that here too. What estimate of $\pi$ does this give?
  4. Now we'll do the same thing, but with a cubic Bézier curve. Figure out which values of $\ell$ make the curve with control points $\mathbf{v}_1 = (1,0)$, $\mathbf{v}_2 = (1,\ell)$, $\mathbf{v}_3 = (\ell,1)$, and $\mathbf{v}_4 = (0,1)$ go through the point $(\sqrt{2}/2,\sqrt{2}/2)$. Include a plot of the resulting Bézier curve, and state the resulting numerical estimate for $\pi$ that this gives?
  5. You can do the following for extra credit. In parts (b) and (c), this is an open question, as far as I can tell—but it's the sort of question that's open because nobody has thought to look it yet, not because it's necessarily especially difficult. It would make for a wonderful paper in this class, that might even be publishable in an academic journal.
    1. The cubic Bézier curve approximation that you found in the above part is actually not the best approximation of the quarter circle in a particular sense. We might say the best approximation of the quarter circle is the one where $|\mathbf{c}(t)|$ is "closest" to $1$, as measured by the min/max: We want to minimize the maximal value of $\big||\mathbf{c}(t)|-1\big|$ over all inputs $t \in [0,1]$. Find the value of $\ell$ that does this, and give it's approximation for $\pi$.
    2. What are the values of $\alpha$, $\beta$, and $\gamma$ if you want to min/max with control points $\mathbf{v}_1 = (\alpha,0)$, $\mathbf{v}_2 = (\beta,\gamma)$, $\mathbf{v}_3 = (\gamma,\beta)$, and $\mathbf{v}_4 = (0,\alpha)$.
    3. What the the control points if we want the best approximation for a degree-$4$ Bézier curve?
